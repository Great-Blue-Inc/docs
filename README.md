# dossiers

Hosted, shareable renders of Great Blue engineering dossiers (single-file HTML, published via the /dossier skill).

- Each dossier lives at `d/<slug>-<token>/index.html` and is served by GitHub Pages.
- Links are unlisted: the token makes the URL unguessable and every page carries a noindex meta tag. Anyone with the link can view. Treat links like secret-gist URLs.
- Updating a dossier overwrites the same path, so a shared link stays current.
